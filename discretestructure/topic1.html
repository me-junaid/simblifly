Fundamentals of Set theory

Set theory is a branch of mathematical logic that studies sets, which are collections of objects. It provides a foundation for most of modern mathematics and is widely used in various areas of the field. Here are some fundamental concepts in set theory:

Sets:
  • A set is a well-defined collection of distinct objects, considered as an object in its own right.
  • Objects in a set are called elements or members.
  • The notation for a set is typically curly braces {}. For example, if A is a set containing the elements 1, 2, and 3, it is denoted as A={1,2,3}
Set Equality:
  • Two sets are equal if and only if they have the same elements, regardless of the order.
  • Example: A={1,2,3} is equal to B={3,2,1}.
Subset and Superset:
  • If every element of set A is also an element of set B, then A is a subset of B, denoted by A⊆B.
  • If A is a subset of B but not equal to B, it is denoted as A⊂B.
  • B is a superset of A, denoted by B⊇A or B⊃A.
Empty Set:
  • The empty set, denoted by ∅ or {}, has no elements.
Cardinality:
  • The cardinality of a set is the number of elements it contains. It is denoted by |A|.
  • Example: If A = {1,2,3},then |A| = 3.
Union and Intersection:
  • The union of sets A and B, denoted by A ∪ B, is the set of elements that are in A, or B, or in both.
  • The intersection of sets A and B, denoted by A ∩ B, is the set of elements that are common to both A and B.
Complement:
  • The complement of set A with respect to a universal set U, denoted by A', consists of all elements in U that are not in A.
Power Set:
  • The power set of a set A, denoted by P(A), is the set of all possible subsets of A, including the empty set and A itself.
Cartesian Product:
  • The Cartesian product of sets A and B, denoted by A×B, is the set of all ordered pairs (a,b) where 'a' is in A and 'b'is in B.
Russell's Paradox:
  • Discovered by Bertrand Russell, this paradox arises when considering the set of all sets that do not contain themselves. It leads to a logical contradiction and highlights the need for careful formulation of set theory axioms.
